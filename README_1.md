
# فلوزا — منصة العمل الحر

فلوزا هي منصة متكاملة للعمل الحر تربط العملاء بالمستقلين المهرة. تم بناؤها بأحدث تقنيات الويب، وتوفر تجربة سلسة لنشر المشاريع، وتقديم العروض، والتواصل الفوري، ومعالجة المدفوعات بشكل آمن.

## المميزات

- **المصادقة** — تسجيل دخول آمن عبر Clerk (بريد إلكتروني أو حسابات اجتماعية)
- **ملفات المستقلين** — صفحات بائع تحتوي على السيرة الذاتية والمهارات والتقييمات
- **قوائم الخدمات** — إمكانية إنشاء وإدارة الخدمات المقدمة
- **نظام الطلبات** — دورة حياة كاملة للطلب من البداية حتى التسليم
- **المراسلة الفورية** — محادثة مباشرة بين العملاء والمستقلين عبر Supabase Realtime
- **التقييمات والمراجعات** — نظام تقييم بالنجوم بعد اكتمال الطلب
- **نظام المتابعة** — إمكانية متابعة المستقلين المفضلين
- **دعم ثنائي اللغة** — واجهة باللغتين العربية والإنجليزية
- **الدفع الإلكتروني** — معالجة المدفوعات المحلية عبر Chargily Pay
- **تطبيق ويب تقدمي** — قابل للتثبيت كـ PWA

## التقنيات المستخدمة

| الطبقة | التقنية |
|--------|---------|
| الإطار | Next.js 14 (App Router) |
| لغة البرمجة | TypeScript |
| التنسيق | Tailwind CSS |
| قاعدة البيانات | Supabase (PostgreSQL + Realtime) |
| المصادقة | Clerk |
| الدفع | Chargily Pay |
| النشر | Vercel |

## طريقة التشغيل

### المتطلبات

- Node.js 18+
- npm أو yarn
- مشروع Supabase
- تطبيق Clerk
- حساب Chargily Pay

### خطوات التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/<enr738>/flowza.git
cd flowza
```

2. تثبيت الحزم:
```bash
npm install
```

3. إعداد متغيرات البيئة:
```bash
cp .env.example .env.local
```

أضف القيم المطلوبة في ملف `.env.local` (راجع `.env.example` للاطلاع على المتغيرات المطلوبة).

4. تشغيل خادم التطوير:
```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## متغيرات البيئة

راجع ملف `.env.example` للاطلاع على قائمة كاملة بمتغيرات البيئة المطلوبة، بما فيها إعدادات Supabase وClerk وChargily Pay.
